## Installation

There's no database except mongo.

```apt-get install libmongoc-dev mongodb libbson-dev crystal build-essentials```

```shards install```

```crystal build src/livepost.cr```

```./livepost```

Optionally get Amber::CMD. Clone amber-crystal/amber_cmd and make.

## Usage

To run the demo:
```
crystal build src/livepost.cr
./livepost
```
